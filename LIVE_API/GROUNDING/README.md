# Grounding with Google Search

This folder contains an example (`grounding.py`) of using **Google Search grounding** with the Gemini API.

## Overview
The script uses the `GoogleSearch` tool to ground Gemini's responses in real-time web search results. This allows the model to answer questions about current events and recent information that may not be in its training data.

## Usage
Run the script using `uv` from the project root:
```bash
uv run LIVE_API/GROUNDING/grounding.py
```
