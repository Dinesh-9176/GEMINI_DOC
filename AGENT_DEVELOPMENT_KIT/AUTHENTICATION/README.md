# Authentication with OpenAPI

This module demonstrates how to configure authentication for agents using OpenAPI tools.

## Features
- **API Key Support**: Uses `token_to_scheme_credential` to handle API key authentication (Query or Header).
- **OpenAPIToolset**: Integrates authenticated credentials into the toolset.

## Example
`openapi_auth.py` shows the configuration process for a YAML-based OpenAPI specification with API key security.
