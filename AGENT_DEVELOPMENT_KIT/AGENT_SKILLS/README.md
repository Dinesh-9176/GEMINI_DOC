# Agent Skills

This module demonstrates how to load and use specialized "skills" from a directory.

## Features
- **Skill Loading**: Dynamically load skill definitions from a structured directory.
- **Skill Toolsets**: Combine multiple skills into a single toolset for an agent.

## Example
The `skill.py` script attempts to load a `weather_skill` and provides it to an agent.

> [!NOTE]
> This example currently uses `google.adk.skills` which may require a specific version or configuration of the `google-adk` package.
