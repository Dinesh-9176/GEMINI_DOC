# Qwen Agent (Ollama)

A minimal example of configuring an ADK agent to use the Qwen model via Ollama.

## Features
- **Local LLM**: Connects to an instance of `qwen3:0.6b` running on Ollama.
- **LiteLlm Interface**: Uses the `LiteLlm` model class for easy integration.

## Usage
`agent.py` provides the basic boilerplate for a helpful assistant using a local model.
