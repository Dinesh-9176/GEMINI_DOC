# Streaming Search Agent

A basic example of a streaming agent that uses external tools to answer questions.

## Features
- **Ollama Integration**: Uses `LiteLlm` configured for `ollama_chat/qwen3:0.6b`.
- **Google Search Tool**: Equips the agent with real-time web search capabilities.
- **Streaming Response**: (Implied) Designed to be used with the ADK streaming interface.

## Usage
The `agent.py` script defines the search agent and its researcher persona.
