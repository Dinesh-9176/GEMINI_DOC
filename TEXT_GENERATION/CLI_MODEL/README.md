# Gemini CLI Model

This folder contains an interactive command-line application (`gemini_cli.py`) that showcases multiple capabilities of the Gemini API in a single script.

## Features
- **Chat with AI:** Engaging in a multi-turn conversation with streaming responses.
- **Analyze Image:** Multimodal capabilities to describe local images.
- **Reasoning Question:** Using the `gemini-3-flash-preview` model's built-in advanced "think" capabilities to answer complex questions.

## Usage
Run the script using `uv` or Python directly from the project root:
```bash
uv run TEXT_GENERATION/CLI_MODEL/gemini_cli.py
```
Follow the interactive prompt to choose your action!
