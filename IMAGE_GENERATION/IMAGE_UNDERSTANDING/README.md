# Image Understanding

This directory contains subfolders demonstrating various ways to use Gemini for image understanding and analysis.

## Folders
- `FILE_API_METHOD`: Passing images via the Google GenAI File API.
- `INLINE_PASSING`: Passing images directly inline.
- `OBJECT_DETECTION`: Using Gemini for detecting objects in images.
- `PROMPTING_MULTI_IMAGE`: Providing multiple images in a single prompt.
- `SEGMENTATION`: Using Gemini for image segmentation tasks.
